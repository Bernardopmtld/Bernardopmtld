# Adivinhe o Número 🎮

Um jogo simples e divertido desenvolvido em **Python** onde o computador escolhe um número aleatório e o jogador deve adivinhá-lo com base em dicas.

## 🚀 Funcionalidades
* Geração de números aleatórios.
* Contador de tentativas.
* Tratamento de erros (validação de entrada).
* Feedback em tempo real (Maior ou Menor).

## 🛠️ Tecnologias
* Python 3.x
* Biblioteca `random` (nativa)
